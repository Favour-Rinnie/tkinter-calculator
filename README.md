# tkinter-calculator
This project is a desktop calculator application built using Python and the Tkinter GUI library. It supports basic arithmetic operations including addition, subtraction, multiplication, and division, with a clean and user-friendly interface.

The application includes input validation and error handling to manage invalid expressions and prevent division by zero. It was developed as a beginner-friendly project to demonstrate core Python concepts such as functions, conditionals, exception handling, and graphical user interface design using Tkinter.

This project is suitable for beginners learning Python GUI development and serves as a foundational example for building interactive desktop applications.
